Práctica Inicial de CSS

Estructura básica de CSS

1.1: CSS significa Cascading Style Sheets, lo que en español significaria hojas de estilo en cascada.

1.2: El CSS sirve para darle estilo y posicionamiento a nuestro proyecto HTML, y para interpretarlo utilizamos los navegadores.

1.3: Se le llama regla de CSS a la forma en la que este mismo le da estilo a alguna etiqueta, debiendo contener un selector, llaves y propiedades.

por ejemplo:

h1 {
color: red;
}

1.4: La primera pregunta la respondí en la consigna anterior.

Selector
Selecciona el elemento HTML que vamos a diseñar.
Llaves
Dentro de estas llaves deben ser ingresadas las declaraciones que se desean agregar, es decir, las propiedades y sus valores.
Propiedades
Se les es asignado un valor para cambiar el diseño de nuestro selector.

1.5: Un selector en CSS nos permite aplicar estilos a elementos especifícos a traves de distintos selectores.

1.6:

Selectores Definición
Selector de ID Este tipo de selector es utilizado para darle un diseño a un unico elemento a través de un identificador.
Selector de Clases Este tipo de selector es utilizado para darle diseño a todos los elementos que posean una misma clase dada por un atributo class.
Selector Universal Permite aplicar un diseño a absolutamente todos los elementos de un proyecto HTML.
Selector de etiquetas Permite aplicar un diseño a una etiqueta en general.
Selector Descendente Este selector permite cambiar el diseño de un elemento que esta dentro de otro elemento.

1.7: El CSS funciona por etapas, teniendo el siguiente ordén

  1. El Navegador carga el HTML.
  2. Se transforma el HTML en DOM, siendo la página representada en la memoria del ordenador.
  3. El Navegador busca los archivos vinculados al documento, incluidos el CSS y Javascript.
  4. El Navegador comienza a analizar el CSS para ordenarlo según sus reglas y selectores utilizados.
  5. El arból de renderización da la estructura en la que debe aparecer la página luego de analizar sus reglas.
  6. Finalmente, la página se muestra en la pantalla.

1.8: Las 3 formas para vincular el CSS con HTML son:

Forma Metodo
Atributo Se le agrega el atributo "style" a una etiqueta para cambiar su diseño.
Etiqueta Se abre una etiqueta "style" y se seleccionan los elementos que se quieren cambiar.
Archivo Se crea un archivo con la extensión .css y se enlaza a nuestro proyecto HTML.
